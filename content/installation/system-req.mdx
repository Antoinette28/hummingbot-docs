Hummingbot has been successfully tested with the following specifications:

Resource | Requirement
---|---
**Operating System** |   **Linux**: Ubuntu 16.04 or later (recommended) *Other Linux installations: Debian GNU/Linux 9, CentOS 7, Amazon Linux 2 AMI, **MacOS**: macOS 10.12.6 (Sierra) or later, **Windows**: Windows 10 or later 
**Memory/RAM** | 1 GB one instance *+250 MB per additional instance*
**Storage** | **Install using Docker**: 5 GB per instance **Install from source **: 3 GB per instance
**Network** | A reliable internet connection is critical to keeping Hummingbot connected to exchanges.
